# email-validator
JavaScript library for email validation

## Usage

```typescript

import { isValidEmail } from 'email-validator';

isValidEmail("test@email.com"); // true


```

## Contribute

Contributions welcome!

## Setup

### Install the dependencies:

```bash
npm install
```

### Build the app:

```bash
npm run build
```

### Test the app:

```bash
npm run test
```
